# Walk-Run Activity Classification

Classifies **walking vs running** from accelerometer and gyroscope data.

## Folder Structure
- data/ (local, ignored)
- notebook/
- output models/ (saved models: .joblib, .h5)

## Quick Start
1. Create venv and install requirements.
2. Open the notebook in `notebook/`.
3. Trained models are saved to `output models/`.
